Human Cannonball
Luke Twyman 05/06/04

Aim: judging wind strength/direction, angle and power, you must complete the 
various stunts in order to gain points, But points are also awarded for failing 
with style! the more splatter, the more points!

Controls:

Ctrl; starts the power meter rising, and stops it again. /menus
spacebar; fire. /menus
arrow keys; view mode, allows you to look over the playing area before firing.

If you experience any lag while playing, close other windows 
and disconnect from the internet. if you have any other problems
email: neverest_songs@yahoo.co.uk

you are advised to Install Universal Black Font before playing if you do not 
already have it installed.
