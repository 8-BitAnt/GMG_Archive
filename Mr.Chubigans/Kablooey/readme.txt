Kablooey! 1.03
by David Galindo, aka chubigans
----------------

Thanks for playing Kablooey! If you're experiencing any technical issues, be sure
to upgrade to the latest Direct X version and be sure any background programs are
disabled. Also, be SURE to unzip the file before playing, or random errors may occur.

System Requirements:

# Windows 2000, Windows XP, or Windows Vista
# DirectX 8 or later
# DirectX 8 compatible graphics card with at least 32MB of video memory
# Pentium or equivalent processor
# DirectX 8 compatible sound card
# 160 MB of memory or greater
# 800x600 (or greater) screen resolution with 16-bit or 32-bit colors
