LIZARDMEN (LIONSOFT, 2006) - INSTRUCTIONS MANUAL

Thanks to have downloaded LIZARDMEN, my new game created with Game Maker (www.gamemaker.nl)  . I hope you will have fun to play this game. If you want to download other Lionsoft games, visit www.lionsoft.altervista.org  . Enjoy!
IMPORTANT - READ BEFORE START TO PLAY

This game don't require installation, before to start the game be sure you have installed the font FORTE included in the zip file, to have a correct visualization of the game informations on the screen.

SYSTEM REQUIREMENT

Lizardmen don't require a large configuration of your PC:

OPERATIVE SYSTEM

WINDOWS XP
PROCESSOR 1 GHZ+
RAM MEMORY 256 MB+
VIDEO MEMORY 16 MB
DISK SPACE 300 MB
PERIPHERALS KEYBOARD

 
LET'S START!

To launch the game, click two times on the game icon. Lizardmen starts to loading (don't worry it's very, very fast!). When the game starts, the Lionsoft logo will appear, next you will find in front of title screen, press ENTER to start! Now you must choice an option by pressing up or down arrow, and select it with ENTER. If you want to load a previous saved game, select CONTINUE, if it's the first time you play Lizardmen, select NEW GAME, a little prologue scene will start.
PROLOGUE SCENES

During game you will find some scenes then show you the story of the game, press ENTER if you want to skip them.

MAP SCREEN
This is the map screen. Let's check it in detail:

1. this is the world map, you can move through the locations by pressing left or right arrow.        2. this cursor indicate where you are in that moment.                                                                        3. this shows the name of the round where you are. Use ENTER, X or Z to start the level with the Lizardmen selected.                                                                                                                              4. these are your personal informations. The number of lifes, your total health points (the max is five) and the number of gold coins you have collected.                                                                       5. these are the fire emblems. There is one in each round and 2 then you can buy in the shop. You must have all the 10 fire emblems to clear the game!                                                                6. this is the character you are using in that moment. There are 3 different lizardmen but you need to save the other two before use them!                                                                                        7. these are the lizardmen abilities. Each lizardmen have special abilities then you must use to reach all the fire emblems!

SAVE SYSTEM

Lizardmen have an automatic save system. Every time you enter in the map screen the game will be saved. To load a previous saved game you need only to select CONTINUE on the title screen!
IN GAME SCREEN

This is the in game screen. Let's check it in detail:

1. these are you currents health points. Every time you be hurted by an enemy or a trap, you will lose an health point. When your healt points reach zero you will lose a life and you will restart the current part of the level.                                                                                                                           2. this is the number of gold coins that you have collected. These are very important in the game, because you can spend that in the shop! So try to get more as possible!                          3. these are the number of lifes then you have. If you reach zero lifes and die you will get the game over!                                                                                                                                              4. this is the current round where you are.
HOW TO PLAY
If you have any trouble after the training, I'm here to explain you the basic movements of Lizardmen:

RIGHT/LEFT ARROW: move Lizardman                                                                                                                                                                               X KEY: jump                                                                                                                                                                                                                              Z KEY: use the flame attack                                                                                                                                                                                               DOWN ARROW (HOLD): crouch (you can use your flame attack when you are crouching!)                                                                                        UP ARROW: when you are in front of a door or a casket use UP to open it!                                                       

Now let's check the special keys:

ESCAPE (ESC): use that during the game to exit the current level (all the items you get will be saved!). Use that in the map screen to exit the game!                                                                                                                                                                                                                                ENTER: use that during the game to make a pause or use it in the other screens to select an option!                                                                     F4: use that to swap from windowed mode and fullscreen mode. In windowed mode you can modify  the dimension of the window.                     M KEY: use that in the map screen to mute the Background Musics during game. Press again to restore the BGMs.                                             F1: show the game information
THE SHOP
When you finish the first round, you can access the shop every times you want. In it, use the UP and DOWN ARROW to select an item and ENTER, Z KEY or X KEY to buy it (if you have enough gold coins, of course!). These are the items then the shop offers:

1. LIFE UP: it costs 50 gold coins and give you an extra life!                                                             2. HEALTH UP: it costs 500 gold coins each and give you an extra health point (there are 3 levels!)                                                                                                                                                        3. EMBLEM: it costs 1000 gold coins each  and you need them to clear the game! (there are 2 fire emblems to buy!)                                                                                                                              4. These are your current gold coins!                                                                                                   5. EXIT: exit the shop and return to the map.
THE BOSS ROUND
There are 3 boss rounds in the game. In these rounds you must defeat the boss by killing him with your flame attack. Each boss have a different types of attacks and different weak point! When you faces a boss on the screen appear a new information:

1. BOSS HP: this indicate the health points of the boss. It starts at 100 and every time you hit the boss, he will lose 10 health points. When the health points reach the zero, the boss is defeated! Now you need to get your reward: a casket will appear, stand in front of it and press UP to open it!
