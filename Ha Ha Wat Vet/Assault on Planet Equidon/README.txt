Assault On Planet Equidon README

Description

Play as an accountant, and slay hordes of ponies to collect their crystals.

Controls

Left / Right arrow keys = Movement
Space = Jump
Z / C = Switch weapons
X = Use weapon
R = Restart level
Esc = Quit
F4 = Toggle full screen mode
F9 = Take screenshot

The basic controls are explained in the game through a tutorial.

Features

- 6 weapons
- 7 levels
- Final boss
