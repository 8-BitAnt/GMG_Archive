uin
by biggt
version 1.00

controls
z charge
x shoot
c toggle
shift push
enter choose
space stats
up jump
down dig
left move
right move
escape return to menu
f4 toggle full screeen
f9 save screenshot

menu
start starts a new game
continue continues a saved game
special look for hints to unlock this option
end ends the game

the game will autosave at certain points. autosaves are indicated by turning gears at the top right of the screen.

if you would be so kind as to report any bugs you may encounter that would be very much appreciated.
